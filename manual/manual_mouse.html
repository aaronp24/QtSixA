Use the Sixaxis as mouse and keyboard



To use a Sixaxis as mouse keyboard, click on the menu “Settings” -> “Configure Sixaxis”.

On the 2nd tab, “Profiles”, click the checkbox “Use Sixaxis as mouse/keyboard”.

Select a Sixaxis profile from the list and click apply.


The next time a Sixaxis gets connected, it will work as a regular joystick plus the keys in the picture.


Using the other options there, you can create your own Sixaxis profiles and add them to the list.

You can also assign a specific profile to a specific Sixaxis, useful if you have many Sixaxis.
