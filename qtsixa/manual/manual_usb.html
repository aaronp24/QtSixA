Connect a Sixaxis through USB



On most Linux systems, connecting the Sixaxis using a regular USB cable works. (You need to press the PS button to activate it)


If you want the accelerometers to work in USB, you can use the “sixad driver on hidraw device” option (in the “Advanced” tab).

Just [re-]connect the Sixaxis over USB (and press the PS button); in QtSixA, press the refresh button.

If you just plugged-in yours Sixaxis, select the last device in the list and click apply.


That will create a new joystick that has the Sixaxis input as defined in Settings.

Unfortunately, the old joystick will still be there, so your Sixaxis will actually control 2 joysticks.


The new joyctick will have the name “PLAYSTATION(R)3 Controller (hidraw)”.
