Other good stuff



There's also other good stuff about QtSixA:


  1. Notifications – show cute notifications when a new device is connected. (in “Configure QtSixA”, “Notify” tab)

  2. Game profiles – auto-modify a game's configuration file to work with the Sixaxis (2nd tab on QtSixA)

  3. Signal quality actions (advanced) – perform Linux shell commands when, for example, the Sixaxis X reaches 70% signal quality (Geeks only - xD)

  4. Start QtSixA at session startup – use it with systray to start QtSixA as a tray icon (in “Configure QtSixA”)

  5. Start QtSixA driver at boot – This option disables bluetooth at boot and starts sixad instead. It connects the devices right away! (in “Configure Sixaxis”, “Advanced” tab)

  6. Check battery Sixaxis – Checks the current battery of the selected Sixaxis (please note that this is a little buggy and doesn't work all the time)
