Translate QtSixA



First of all, to make sure you don't start translating an already translated language, mail me (falktx@gmail.com)


Translate QtSixA is very easy. Here are the steps:

1 - Get version >= 1.2.0 and install it (older versions don't support translations)

2 - Run “qtsixa” in a terminal. One of the printed messages should say: “locale is: yy_ZZ” (yy_ZZ is your system language)

3 - Copy the file “/usr/share/qtsixa/lang/default.ts” somewhere and rename to “yy_ZZ.ts” (same name as in the step before)

4 - Install the application “Qt4 Linguist” (in Ubuntu use 'sudo apt-get install qt4-dev-tools')

5 - Run “Qt4 Lingust” and open the file you just copied.

6 - Translate the application


Once the translation is completed (or at least most of it), send the *.ts file to my mail.


And that's it.



Happy translation!
